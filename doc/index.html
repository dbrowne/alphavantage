alphavantage

Alpha Vantage Rust client — API documentation for the workspace

Core

av-core core

Shared types, configuration, error handling, and API function identifiers used by all crates.

av-models models

Strongly-typed serde models for all Alpha Vantage API responses: time series, fundamentals, news, forex, crypto.

Data Access

av-client client

HTTP client for the Alpha Vantage REST API with rate limiting and response parsing.

av-database-postgres database

Diesel ORM models, repository pattern, and TimescaleDB integration for market data storage.

Ingestion

av-loaders loader

Data ingestion pipelines: symbol listing, overviews, intraday/summary prices, news sentiment, top movers.

crypto-loaders loader

Multi-provider crypto data aggregation: CoinGecko, CoinMarketCap, CoinPaprika, CoinCap, SosoValue.

Command-line interface for running loaders, querying data, and managing the database.
